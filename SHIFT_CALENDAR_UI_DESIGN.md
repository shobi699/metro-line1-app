## ۱) جهت‌گیری طراحی (Design Direction)

**موضوع:** تقویم برای راهبر متروست — کسی که زندگی‌اش روی «خط» می‌گذرد. پس زبان بصری تقویم از دنیای خود مترو می‌آید:

> **امضای طراحی (Signature): «خطِ زندگی» (Life Line)**
> روزها مثل **ایستگاه‌های یک خط مترو** روی یک ریبون پیوسته می‌نشینند؛ رنگ هر قطعه از خط = نوع شیفت آن روز؛ نشانگر «امروز» = **قطاری که روی خط ایستاده**. شیفت شب که از نیمه‌شب عبور می‌کند، مثل **تونل** دو ایستگاه (دو روز) را به هم می‌دوزد.

این استعاره در سه جا زندگی می‌کند: نوار هفته بالای نمای روزانه، هدر نمای هفتگی، و ویجت متوسط موبایل. **بقیه‌ی تقویم آرام و منضبط می‌ماند** — گرید ماهانه تمیز، بدون تزئین، تا امضا برجسته بماند (خرج جسارت فقط در یک جا).

اصول:
- **یک نگاه، یک پاسخ:** مهم‌ترین سوال کاربر «امروز/فردا چه شیفتی‌ام؟» است — پاسخ همیشه در بالاترین سلسله‌مراتب بصری.
- **رنگ = معنا، نه تزئین:** رنگ فقط برای کد شیفت و وضعیت. هر رنگ همیشه با آیکون + برچسب همراه است (کوررنگی).
- **قرمز برند مقدس است:** قرمز `brand-500` فقط برای «امروز»، تعطیل رسمی و CTA اصلی — هرگز برای شیفت (تا حس هشدار ندهد).

## ۲) توکن‌های اختصاصی تقویم (افزوده به Design System)

### ۲.۱ پالت شیفت (Shift Palette) — روشن / تیره
انتخاب‌شده برای تمایز کوررنگی‌امن (آبی/کهربایی/بنفش/سبز محورهای متمایز) و آرامش سازمانی:

```
--shift-morning:  #0E9F9F / #2DD4CF   ☀️ صبح    (فیروزه‌ای — آسمانِ آغاز روز، نه زردِ هشدار)
--shift-evening:  #D97706 / #FBBF24   🌆 عصر    (کهربایی غروب)
--shift-night:    #4F46E5 / #818CF8   🌙 شب     (نیلی عمیق)
--shift-off:      #16A34A / #4ADE80   🏖 آف     (سبز آسودگی)
--shift-office:   #64748B / #94A3B8   🏢 اداری  (خاکستری-آبی خنثی)
--shift-forecast: هر رنگ با opacity 45% + حاشیه dashed   (پیش‌بینی سیکل، هنوز لوحه نیامده)
```

هر شیفت یک **جفت** دارد: `-bg` (تینت ۱۰٪ برای پس‌زمینه خانه) و رنگ اصلی (نوار/متن). این رنگ‌ها در پنل ادمین قابل بازتعریف‌اند — این‌ها پیش‌فرض کارخانه‌اند.

### ۲.۲ لایه‌های رویداد

```
--evt-personal: #2563EB (info)   ● نقطه/چیپ رویداد شخصی
--evt-org:      #7C3AED          ◆ رویداد سازمانی (لوزی برای تمایز شکلی)
--evt-task:     #0891B2          ☐ کار (مربع قابل تیک)
--holiday:      brand-500        عدد روز قرمز + زمینه brand-50
```

### ۲.۳ تایپوگرافی تقویم

```
عدد روز (ماهانه):     17/24  600   fa-IR
عدد روزِ امروز:        17/24  700   سفید روی دایره brand-500
عنوان ماه:            Display 28/36 700  «مرداد ۱۴۰۳» + زیرنویس میلادی Caption
برچسب شیفت در خانه:   Caption 12/16 500
ساعت در آژندا:        Mono (LTR isolate) 13/20
```

## ۳) UI وب — صفحه‌به‌صفحه

### ۳.۰ شِمای کلی صفحه (Layout)

```
┌────────────────────────────────────────────────────────────────┐
│ ◀ تیر   « مرداد ۱۴۰۳ »   شهریور ▶     [ماه|هفته|روز|سال]  [+ رویداد] │ ← هدر چسبان
│ July–Aug 2024 (Caption faint)                    [⚙ لایه‌ها ▾]      │
├───────────────────────────────────────────┬────────────────────┤
│                                           │  پنل «امروز» (چسبان) │
│           گرید ماهانه ۷×۵                 │  ┌──────────────┐  │
│                                           │  │ 🌙 شب        │  │
│                                           │  │ ۲۲:۰۰–۰۶:۰۰  │  │
│                                           │  │ قطار ۱۱۰      │  │
│                                           │  └──────────────┘  │
│                                           │  ● دکتر ۱۷:۰۰      │
│                                           │  ☐ مدارک بیمه      │
│                                           │  ─────────────     │
│                                           │  ۲ روز تا آف 🎉    │
│                                           │  [تعویض شیفت]      │
└───────────────────────────────────────────┴────────────────────┘
```

- ستون کناری در ≥1024px؛ زیر آن، پنل «امروز» به بالای گرید منتقل می‌شود (Card افقی).
- هدر sticky با سایه‌ی ظریف هنگام اسکرول (`shadow-sm`).

### ۳.۱ خانه‌ی روز در نمای ماهانه (Anatomy of a Day Cell)

```
┌──────────────┐   قواعد:
│ ۱۵        ◆  │   • عدد روز بالا-راست (RTL). جمعه/تعطیل: قرمز.
│ ☀️ صبح       │   • نوار شیفت: پسِ‌زمینه تینت + آیکون + برچسب — نه فقط رنگ.
│ ●●           │   • حداکثر ۲ نقطه رویداد + «+۳» اگر بیشتر بود.
│              │   • امروز: حلقه‌ی brand-500 دور خانه + عدد در دایره پر.
└──────────────┘   • پیش‌بینی: نوار شیفت کم‌رنگ + حاشیه dashed + tooltip «پیش‌بینی سیکل».
                   • تعویض در انتظار: آیکون ⇄ چشمک‌زن ملایم گوشه خانه.
```

Hover (وب): خانه `surface-hover` + Popover خلاصه (شیفت، اعزام لوحه، رویدادها) بعد از 400ms. کلیک: باز شدن **پنل روز** (Drawer از راست) — نه پرش صفحه.

### ۳.۲ نمای هفتگی — خانه‌ی امضا
بالای ستون‌ها، **«خطِ زندگی»**: ریبون پیوسته‌ی ۷ قطعه‌ای با رنگ شیفت هر روز، ایستگاه‌ها = دایره‌های کوچک با عدد روز، قطار کوچک روی ایستگاه امروز. زیر آن، ستون‌های ساعتی (۰۰–۲۴) که بلوک شیفت به‌صورت زمان واقعی رسم می‌شود:

```
   خطِ زندگی:  ──●━━☀━━●━━☀━━●━━🌆━━●━━🌙━━🚇━━🏖━━●━━🏖━━●──  ← قطعه‌ی 🌙 با گرادیان
                ش      ی      د      س     چ(امروز) پ      ج        وارد ستون پنجشنبه می‌شود (تونل)
```

بلوک شیفت شب از ۲۲:۰۰ چهارشنبه تا ۰۶:۰۰ پنجشنبه **یکپارچه** رسم می‌شود (گوشه‌های گرد فقط ابتدا/انتها) — حل بصری مشکل کلاسیک شیفت شب.

### ۳.۳ نمای روزانه / آژندا
تایم‌لاین عمودی: خط عمودی نازک + گره‌ها. بلوک شیفت بزرگ (با جزئیات لوحه: قطار، مسیر، اعزام‌ها اگر لایه روشن باشد)، رویدادها با کارت‌های کوچک رنگ‌لایه، کارها با چک‌باکس درجا. خط «اکنون» قرمز باریک با نقطه تپنده (اگر روز = امروز).

### ۳.۴ نمای سالانه
۱۲ مینی‌ماه (گرید ۴×۳). هر روز = مربع ۱۰px با رنگ شیفت (Heatmap زندگی). تعطیلات = نقطه قرمز. **بنر پل تعطیلات** بالای صفحه: کارت‌های افقی اسکرول‌شونده «✈️ ۱۲–۱۵ شهریور · ۴ روز» با دکمه «برنامه بریز». Hover روی هر بازه ← هایلایت همان روزها در مینی‌ماه‌ها (اتصال بصری).

### ۳.۵ پنل روز (Drawer)
از راست باز می‌شود (RTL)، عرض 400px: هدر تاریخ دوگانه، کارت شیفت (+ دکمه «درخواست تعویض»)، بخش لوحه (اگر منتشر شده)، لیست رویدادها/کارها با افزودن سریع درجا (input همان‌جا، بدون مودال)، دکمه «افزودن یادآور». ESC یا کلیک بیرون = بستن.

## ۴) UI موبایل (Expo) — صفحه‌به‌صفحه

### ۴.۰ ساختار تب «تقویم»

```
┌──────────────────────────────┐
│ « مرداد ۱۴۰۳ »        [⚙][≡] │ ← هدر مینیمال؛ لمس عنوان = پرش به امروز
│ ش  ی  د  س  چ  پ  ج          │
│ گرید ماهانه (خانه‌های ۴۸px+) │ ← سوایپ افقی = ماه قبل/بعد (فنری)
│  ...                         │
│ ─────── دستگیره ───────      │
│  Bottom Sheet «روز منتخب»    │ ← همیشه حاضر؛ سه لنگر: بسته/نیمه/تمام‌صفحه
│  🌙 شب ۲۲:۰۰–۰۶:۰۰ · قطار ۱۱۰│
│  ● دکتر ۱۷:۰۰   ☐ مدارک     │
│                        [＋]  │ ← FAB افزودن سریع
└──────────────────────────────┘
```

- **الگوی کلیدی:** به‌جای ناوبری صفحه‌ای، **Bottom Sheet سه‌لنگری** — لمس هر روز، شیت نیمه‌باز محتوای همان روز را نشان می‌دهد؛ کشیدن بالا = جزئیات کامل + تاریخچه؛ گرید همیشه در دید می‌ماند (حس «تقویم واقعی روی میز»).
- لمس طولانی روز ← منوی زمینه: «رویداد جدید / کار جدید / درخواست تعویض این شیفت».
- نمای هفتگی موبایل = همان «خطِ زندگی» به‌صورت نوار افقی اسکرول‌شونده بالای آژندای عمودی.
- ژست‌ها: سوایپ افقی (ماه)، Pinch روی گرید (ماه ⇄ سال)، Pull-to-refresh با انیمیشن قطار کوچک 🚇 که رد می‌شود (لحظه‌ی لذت، محترم به reduced-motion).

### ۴.۱ فرم افزودن سریع (Bottom Sheet)
یک فیلد بزرگ «چی و کی؟» + چیپ‌های نوع (رویداد/تولد/کار) + چیپ‌های زمان سریع (امروز، فردا، آف بعدی ← هوش شیفتی!) + یادآور با سه گزینه‌ی پیش‌فرض. ثبت در ≤۳ لمس. تاریخ‌یاب جلالی چرخ‌فلکی (drum picker) بومی.

### ۴.۲ حالت‌های خاص
- **آفلاین:** بنر خاکستری «آفلاین — تقویم تا ۳ ماه در دسترس است»؛ همه‌چیز کار می‌کند؛ افزودن‌ها صف می‌شوند (نشان ساعت روی چیپ).
- **تداخل:** هنگام ثبت رویداد روی روز شیفت، کارت هشدار زرد داخل همان فرم (نه مودال جدا) + دکمه ثانویه «درخواست تعویض».

**قواعد ویجت:** حداکثر ۲ فونت‌سایز در هر اندازه؛ بدون اسکرول؛ حالت «داده کهنه» (>۲۴h همگام‌نشده) = نقطه زرد گوشه؛ حالت خطا = «برای همگام‌سازی باز کنید» با آیکون، نه متن قرمز.

## ۶) میکروتعامل‌ها و حرکت (Motion)

| لحظه | انیمیشن | مدت |
| :--- | :--- | :--- |
| تغییر ماه | اسلاید افقی جهت‌دار + fade اعداد | base 200ms |
| باز شدن پنل/شیت روز | spring ملایم (بدون bounce زیاد) | slow 320ms |
| تیک کار | چک‌باکس پر می‌شود + خط خوردن متن + haptic سبک (موبایل) | fast 120ms |
| تایید تعویض شیفت | رنگ خانه‌ی روز با cross-fade عوض می‌شود + پالس تک‌ضرب حلقه | base |
| نشانگر «اکنون» | نقطه تپنده آرام (scale 1→1.15، هر 2s) | ambient |
| Pull-to-refresh | قطار 🚇 از راست به چپ رد می‌شود | 600ms |

همه با `prefers-reduced-motion` خاموش/ساده می‌شوند (cross-fade جایگزین اسلاید).

## ۷) حالت‌های سیستم (هیچ صفحه‌ای بدون این‌ها منتشر نمی‌شود)

- **Skeleton:** گرید ماه = ۳۵ مستطیل تپنده با نوارهای شیفت خاکستری؛ پنل امروز = ۳ خط.
- **خالی (روز بدون چیز):** «روز آزاد شماست ✨» + دکمه «رویدادی اضافه کنید» — دعوت به عمل، نه سکوت.
- **خالی (لوحه نیامده):** نوار شیفت پیش‌بینی + Caption «بر اساس سیکل — لوحه هنوز منتشر نشده».
- **خطا:** بنر داخل صفحه «تقویم به‌روز نشد — آخرین نسخه‌ی ذخیره‌شده را می‌بینید» + دکمه «تلاش دوباره». هرگز صفحه‌ی سفید.
- **دسترسی‌پذیری:** ناوبری کیبورد کامل روی گرید (جهت‌نما + Enter = پنل روز)، focus ring مرئی، aria-label هر خانه («شنبه ۱۵ مرداد، شیفت شب، ۲ رویداد»)، کنتراست AA در هر دو تم، هدف لمسی ≥44px.

## ۸) سناریوهای تجربه (UX Scenarios)

### سناریو ۱ — «ثانیه‌ی اول»
احمدی تب تقویم را باز می‌کند. قبل از هر اسکرول: پنل «امروز» با کارت شیفت بزرگ (🌙 شب، قطار ۱۱۰) بالای همه است و در گرید، امروز با حلقه قرمز می‌درخشد. سوالش («امشب چی‌ام؟») در <۱ ثانیه و بدون هیچ لمسی جواب گرفته.

### سناریو ۲ — ثبت ۳ لمسی
لمس طولانی روی پنج‌شنبه ← «کار جدید» ← می‌نویسد «مدارک بیمه» ← چیپ یادآور «صبح همان روز» ← ثبت. شیت بسته می‌شود، مربع ☐ روی خانه پنج‌شنبه با انیمیشن ظریف می‌نشیند.

### سناریو ۳ — فهم شیفت شب با یک نگاه
رضایی نمای هفته را باز می‌کند. بلوک نیلی شب از چهارشنبه ۲۲:۰۰ شروع شده و بدون بریدگی وارد ستون پنجشنبه تا ۰۶:۰۰ شده؛ روی «خطِ زندگی» بالای صفحه هم قطعه‌ی نیلی با نماد تونل 🚇 دو ایستگاه را دوخته. برای اولین بار «شب دوشنبه یعنی کی می‌خوابم؟» بصری جواب دارد.

### سناریو ۴ — ویجت در آسانسور
کاربر در آسانسور فقط گوشی را روشن می‌کند: ویجت متوسط، ریبون هفته را نشان می‌دهد — دو آف سبز آخر هفته. لبخند. لمس روی آف ← اپ روی همان روز با شیت نیمه‌باز.

### سناریو ۵ — شخصی‌سازی ادمین، بازتاب همه‌جا
ادمین رنگ شیفت عصر را نارنجی‌تر و برچسبش را «بعدازظهر» می‌کند. وب، موبایل، ویجت (در همگام‌سازی بعدی)، خروجی چاپی و ICS همه از همان توکن settings می‌خوانند — هیچ‌جا ناهماهنگ نمی‌ماند.

## ۹) نقشه پیاده‌سازی فرانت

| گام | محدوده |
| :--- | :--- |
| ۱ | توکن‌های شیفت در `@theme` (وب) و `theme.ts` مشترک (موبایل) + کامپوننت‌های پایه: `DayCell`, `ShiftBadge`, `EventDot`, `LifeLine` |
| ۲ | وب: `MonthGrid` + پنل امروز + Drawer روز + نمای هفته (رندر بلوک دو-روزه شب) |
| ۳ | موبایل: گرید + Bottom Sheet سه‌لنگری + افزودن سریع + ژست‌ها |
| ۴ | نمای سال + پل تعطیلات + آژندا + حالت‌های skeleton/خالی/خطا |
